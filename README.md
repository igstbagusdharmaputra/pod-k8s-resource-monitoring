# pod-k8s-resource-monitoring

### Feature
- Generate data in console way for CPU and memory resource usage and request

### How to Use
```
go run main.go -kubeConfig <File Confi Kube> -namespace <Namespace>
example : go run main.go -kubeConfig ~/.kube/config -namespace kube-system
```
